Virtual Select - Server-Side Pagination Test

How to test:

  1. Open the dropdown by clicking on it
  2. The first page (50 options) will load automatically
  3. Scroll down to the bottom of the options list
  4. The next page will load automatically when you reach the bottom
  5. Continue scrolling to load more pages
  6. Check the log below to see the pagination events
  7. Try searching - notice how totalItems resets and is recalculated
  8. Clear search - totalItems resets again for new query

Optimization:

The totalItems is calculated only on the first page request and passed to subsequent pages. When the search changes, totalItems resets to 0, forcing a recalculation. This avoids expensive COUNT(*) queries on every pagination request.
